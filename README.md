# The Grand IPA Archive
Forked from relikd, indexed even more app.

NO APPS ARE HOSTED ON THIS SITE.

Hello, this project aims to provide a searchable and filterable index for .ipa files. None of the linked files are mine, nor am I involved in any capacity on the referenced projects. I merely forked the crawler to index IPA files in various [Archive.org](https://archive.org/) collections. The list of indexed collections can be found at data/urls.json.
